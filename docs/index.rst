Moonpie: Love2D Framework
===================================

Moonpie is a framework designed for easy development within the Love2D engine.
It is focused around providing great UX with responsive designs for flexible layouts of components. 
At the same time it allows for flexibility in how components are rendered. There is also support
with libraries for collections, math, entity-component-systems, etc...
